# Employee Trends Analysis (Tableau)

## Overview
This project features an interactive Tableau dashboard created to analyze HR data and
derive insights into employee demographics, workforce distribution, satisfaction, and
attrition trends. The dashboard enables exploratory analysis to support data-driven
HR and workforce planning decisions.

## Objectives
- Analyze employee distribution across departments and roles
- Identify demographic trends related to age, gender, and education
- Evaluate employee satisfaction and engagement metrics
- Examine factors associated with employee attrition and turnover
- Support data-driven decision-making for HR initiatives

## Tool Used
- Tableau

## Dataset
The dataset contains employee-level HR information, including:
- Department and job role
- Gender and age
- Education field and level
- Job satisfaction
- Travel frequency
- Attrition status

## Dashboard Description
The dashboard consolidates employee data into a unified view and allows users to
interactively explore trends by department, demographics, and job attributes using
filters and visualizations.

## Key Visualizations
- Headcount and demographic distribution
- Employee distribution by department and job function
- Education field vs average job satisfaction
- Travel frequency analysis
- Attrition-related views
- Core HR KPIs and workforce metrics

## Use Case
This dashboard can be used by HR and business teams to understand workforce
composition, monitor satisfaction trends, and identify potential risk areas related
to employee attrition.
